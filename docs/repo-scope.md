# Repository Scope

This repository documents the current reverse-engineering state of *Lands of Lore II* and publishes the cleaned closure notes needed to review the LoL2 lane without the full private working ledger.

Included:

- promoted reverse-engineering writeups
- evidence indexes and witness maps
- tool notes and workflow summaries
- future patch planning notes kept separate from canon

Excluded from the initial repo:

- raw retail game files
- full copyrighted asset dumps
- raw AI memo sprawl
- one-off scratch scripts that only served a single local test

The goal is the same standard used for the LoL1 repo:

- readable
- reviewable
- technically honest
- useful without turning the repo into a copyrighted asset mirror
