# Evidence

This folder is for evidence indexes and artifact references.

Current contents:

- `lol2-evidence-index.md`
  - Where the supporting local trace/artifact files live.
- `lol2-witness-map.md`
  - Which witness or trace variant proved which part of the current model.

Intended contents:

- trace inventory
- witness mapping
- links to large local artifacts kept outside the repo when needed
- small supporting summaries that explain what a trace proves

Do not treat this folder as the place for final conclusions.
